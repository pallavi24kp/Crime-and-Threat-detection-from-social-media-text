About This System

ML Model

TF-IDF vectorization with multi-label Logistic Regression trained on the Jigsaw Toxic Comment Classification dataset. Detects 6 distinct threat categories simultaneously.

6 Threat Categories

Toxic, Severely Toxic, Obscene, Direct Threats, Personal Insults, and Identity-based Hate. Each category is scored independently with a confidence percentage.

Real-Time API

Flask REST API with CORS support. Endpoints for single-text analysis, batch processing (up to 50 texts), health monitoring, and model statistics.

BERT-Ready

The system includes crime_detection.py for full BERT fine-tuning. Once trained, BERT weights can replace the lightweight model for state-of-the-art accuracy.

API Reference

GET /api/health Service health check
GET /api/stats Model statistics and label definitions
POST /api/analyze Analyze a single text → {"text": "..."}
POST /api/batch Batch analyze up to 50 texts → {"texts": [...]}
Example Request 
curl -X POST http://localhost:5000/api/analyze \
  -H "Content-Type: application/json" \
  -d '{"text": "I will make you pay for this!"}'
